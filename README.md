# SmartHome

Just a first project to learn the basics of some interfaces

## Possibilities

The project displays a greeting and three readings on the display screen:
- illumination
- temperature
- humidity

## Components

- STM32F401RCT6
- IPS TFT ST7735S SPI 80x160
- illumination sensor BH1750 I2C
- temperature and humidity sensor GY-21 I2C
